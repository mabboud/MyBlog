I started a project MySwing on java.net which will consolidate my work from this weblog. This project is under Lesser General Public License (LGPL)

Current Status of Project:

  1. Incremental Search: 
         Complete implementation of incremental search along with two bug fixes.
              Bug 1: component doesn't get focus when search popup gets closed in case
                        the popup is heavy weight
              Bug 2: search in JEditorPane with some html doesn't work.
         An updated webstart demo is also available in homepage.
     
  2. Resizable Table:
          A JTable extension which allows resizing of both rows and columns
          just simply by dragging the grid lines. The component name is MyTable.

Any custom components in this project will be prefixed by "My". For example the JTable extension will be MyTable.

There are two jars (myswing.jar & myswing-demo.jar) available for download. and the homepage contains the webstart demoes for the project.

You can get the sources directly from CVS.
