Andy Roberts found an interesting bug:
   drag the toolbar to the left or right of the challenge, so that it's vertical. You'll notice that the arrow button is no longer adjacent to its associated main button

now it is fixed. Changes are done in addToToolBar method:

    public JButton addToToolBar(JToolBar toolbar){ 
        JToolBar tempBar = new JToolBar(); 
        tempBar.setAlignmentX(0.5f);
        tempBar.setRollover(true); 
        tempBar.add(mainButton); 
        tempBar.add(arrowButton); 
        tempBar.setFloatable(false); 
        toolbar.add(tempBar); 
        return mainButton; 
    }
